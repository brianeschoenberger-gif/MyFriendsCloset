# My Friends Closet

A private social wardrobe app where friends share, borrow, and style each other's clothes.

> Shop your friends' closets before you buy something new.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite. On Windows PowerShell systems with script execution disabled, use `npm.cmd` instead of `npm`.

## Checks

```bash
npm run lint
npm run build
```

## Current prototype

The mock-data prototype includes a responsive dashboard, My Closet with add-item and visibility controls, friends' closet browsing, borrow requests, mock AI outfit generation, and a Palm Springs event closet.
